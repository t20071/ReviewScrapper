# ReviewFlask
In this project, scraping the products' reviews from the Flipkart website in the CSV file format with the column name (Product, Customer Name, Rating, and Heading Comment) have performed.


**Libraries used**
1. Flask
2. requests
3. beautifulsoup4
4. click
and many more
